# Tworzenie aplikacji bazodanowych - Projekt

## Temat projektu

Multitematyczna platforma e-learningowa z modułem e-commerce i funkcjami społecznościowymi

### Opis projektu

Przedmiotem projektu jest stworzenie aplikacji webowej dla firmy prowadzącej szkolenia. System będzie realizował funkcje sklepu internetowego umożliwiającego przeglądanie oferty szkoleniowej oraz zakup kursów i dostęp do powiązanych z nimi elektronicznych materiałów dydaktycznych

### Technologie

| Kategoria       | Technologia / Narzędzie | Zastosowanie                                                  |
| --------------- | ----------------------- | ------------------------------------------------------------- |
| **Frontend**    | React + typescript      | Warstwa wizualna                                              |
| **Backend**     | C# / .NET               | Logika biznesowa, kontrolery                                  |
| **Baza danych** | PostgreSQL              | Przechowywanie danych o kursach, użytkownikach i zamówieniach |
